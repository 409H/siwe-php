# SIWE PHP
### Port of Siwe library in PHP (Sign In With Ethereum)

### Install
```bash
composer install iltumio/siwe-php
```

## Development
### Install dependencies
```bash
composer install
```

### Run tests
```bash
./vendor/bin/phpunit
```
## Disclaimer
This library is **released as is** and has not undergone any formal security audits.
